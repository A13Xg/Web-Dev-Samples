## Project Overview

**Web Dev Samples** is a curated collection of 13 fully interactive, self-contained website templates built with pure HTML, CSS, and JavaScript — no frameworks, no build tools, no external dependencies. Each template demonstrates a distinct visual identity and includes real working interactions (forms, modals, carts, search, etc.).

The flagship template, **The Theme Atlas** (Site 13), is a comprehensive CSS/design-theme reference library showcasing **65 named design themes**, each with a live sample block, color palette swatches, category filtering, and a one-click copy-CSS button.

## Theme Categories

The Theme Atlas supports seven category filters accessible via the filter navigation at the top:

| Category | Filter Value | Tag Color |
|----------|-------------|-----------|
| All | `all` | — |
| Dark | `dark` | Cyan |
| Light | `light` | Yellow |
| Retro | `retro` | Pink |
| Modern | `modern` | Indigo |
| Artistic | `artistic` | Purple |
| Minimal | `minimal` | Grey |

Clicking a category button hides all cards that don't belong to that category using a `.hidden` CSS class toggled via JavaScript.

## Using the Copy-CSS Feature

Each theme card has a **⌘ Copy CSS** button. Clicking it:

1. Looks up a pre-written CSS snippet in the JavaScript `snippets` object (keyed by theme number).
2. Writes the snippet to the clipboard using `navigator.clipboard.writeText()`, falling back to `document.execCommand('copy')` for older browsers.
3. Shows a toast notification ("✓ Copied to clipboard!") for 2.2 seconds.

The copied CSS contains the minimum set of class declarations needed to apply the theme's colours and core properties. You can paste it into any project and use the listed class names on your own HTML elements.

**Example — Vampire theme CSS:**
```css
.vampire {
  background: linear-gradient(135deg, #0a0000, #1a0005);
  border: 2px solid #8b0000;
  border-radius: 8px;
  color: #dc143c;
  font-family: "Playfair Display", serif;
  text-shadow: 0 0 12px rgba(220, 20, 60, .6);
}
.vampire-btn {
  background: linear-gradient(135deg, #8b0000, #c0000a);
  border: 1px solid #dc143c;
  box-shadow: 0 0 10px rgba(220, 20, 60, .35);
}
```

## Local Development

**Requirements:** Node.js v14+ and npm

```sh
# Clone
git clone https://github.com/A13Xg/Web-Dev-Samples.git
cd Web-Dev-Samples

# Install Express
npm install

# Start dev server on http://localhost:3000
npm start
```

Open **http://localhost:3000** in your browser. All 13 templates are served from the `/docs` folder.
