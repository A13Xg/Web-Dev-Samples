Alpha 0.3.7 — Co-Op Netcode Overhaul

We've completely rewritten the multiplayer netcode using a custom rollback system. Latency dropped from ~120ms to sub-40ms in our internal tests. The new ghost-data system for dead players is also live in this build — your run isn't truly over until it is.

Procedural City Gen: How We Build Sector 7

Deep-dive into our wave function collapse algorithm powering city generation. Each district has a unique "vibe graph" that controls architecture density, lighting palettes, and enemy faction spawns. This took 3 months to get right.
